# murphy_msgs

Placeholder package for Murphy-specific ROS interfaces.

When you add `.msg`, `.srv`, or `.action` files, declare them here with `rosidl_generate_interfaces` and depend on `rosidl_default_generators` / `rosidl_default_runtime` per the [ROS 2 interface docs](https://docs.ros.org/en/rolling/Tutorials/Beginner-Client-Libraries/Custom-ROS2-Interfaces.html).
