## What is MoveIt?

MoveIt is an open source robotics manipulation framework for developing new applications, prototyping designs, and benchmarking algorithms.

Are you a commercial company looking to quickly deploy robots to production? Check out [MoveIt Pro](https://picknik.ai/pro).

## PickNik's Stewardship of MoveIt

PickNik was founded with the mission of providing powerful platforms for the robotics industry. MoveIt OSS is our original product, though notably MoveIt was originally conceived at Willow Garage. PickNik has since developed a professional, more mature version of MoveIt called [MoveIt Pro](https://picknik.ai/pro).

While PickNik leads the growth and maintenance of MoveIt, we believe it's important to acknowledge the vibrant **international community of collaborators and maintainers** that play a huge role by improving the software and adding new features.


![hero image](https://picknik.ai/assets/images/hero2.png)

##### Business Model

PickNik is a for-profit company that balances the need to improve the open source code of ROS and MoveIt with the need to develop commercial software - [MoveIt Pro](https://picknik.ai/pro).


##### Stewardship

**What PickNik does to improve the MoveIt project:**

- Provides engineers that review pull requests, fix regressions, and add new features to the open source.

- Provides organizational leadership, outreach, and publicity.

- Organizes many events throughout the year, including World MoveIt Day and MoveItCon.

- Mentors interns and Google Summer of Code students to improve the code.

- Runs monthly and weekly video call working groups for MoveIt developers.

- Contributes portions of code from commercial projects back to improving MoveIt and [ROS](https://picknik.ai/ros/).

- Makes binary/debian releases at intervals throughout the year.


##### Our Open Source Commitment

**We promise that:**

- We won't remove features from the open source codebase in order to make the same feature paid (features might be removed from the open source codebase due to changes in the application).

- We will always release and open source all tests that we have for an open source feature.

- The open source codebase will not contain any artificial limits (performance, etc.).

- We will always allow you to benchmark the performance of MoveIt.
